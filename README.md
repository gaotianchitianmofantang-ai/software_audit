# ソフトウェア監査システム

## 概要

このシステムは、社内で利用するOSSソフトウェアの監査を自動化します。

- **脆弱性チェック**: JVN脆弱性対策情報データベースとの連携
- **更新頻度確認**: GitHubリポジトリの活動状況確認
- **例外申請管理**: 承認フローの自動化
- **レポート生成**: 監査結果の詳細レポート作成
